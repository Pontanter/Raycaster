# Java Raycaster
### A simple Doom-style Raycaster made using Java (swing)

*Controls*
 * **WASD & Arrows for motion**
 * **Space - Jump**
 * **Shift - Dash**
 * **F1 - Toggle debug mode**
 * **F2 - Toggle view mode**

*Minor features include;*
 - Wall jumping
 - Camera shake (footsteps and such)
 - Mirrors
